模块化开发

全新的架构和模块化的开发机制，便于灵活扩展和二次开发。

强大的文档模型

通过和某个文档模型绑定，不同分类可以实现差异化的功能，轻松实现诸如资讯，下载，讨论和图片等功能。

代码完全开源
